# PAPERS

Feel free to check out some of my research papers, as well as reports I have written for several projects. 

## Research Papers

* Antivirus Software: Conducted during ECE1155 (Information Security). This research paper discusses topics related to cybersecurity including several methods used to prevent virus attacks such as signature and heuristic analysis. Additionally, Machine Learning approaches used for different operating systems are described.

* Big Data In Medicine: Conducted in 2018 as part of the TPE (Travaux Pratiques Encadrés) for the French Baccaulerate. We researched Big Data in Medicine, specifically how AI and Machine Learning can be used to predict diseases based on genetic ancestry and symptoms, as well as how robots can be used to assist in surgeries and telemedicine. Only the Final Report is provided here. For more info please visit the "projects" section of my GitHub page. Also note that the paper is written in French, but the general gist of it can be understood through subtitles and figures.
  
* Wearable System for Positioning Ultrasound Scanner: Conducted during Summer Undergraduate Research Program at the Chinese University of Hong Kong. This paper discusses robotics (wearable systems, soft robots, autonomous systems, origami, etc.) used in Ultrasound Scanning. Additionally, several Deep Learning methods employed to localize targetted areas in soft tissues using Ultrasound Scanners are featured. Worked with a team of graduate students and with my supervisor Dr. Hongliang Ren. Paper is also published on MDPI! 

## Project Reports

* Data Science Project: Conducted during STAT1261 (Principles of Data Science). This report studies the effects of several factors such as age, sex, smoking, diabetes, and blood platelets on deaths by heart failures. Several techniques in R are used to analyze and visualize the dataset (from Kaggle). Various Machine Learning models were trained to find the best optimal accuracy.
  
* Dispose It Game: Conducted during ECE1895 (Junior Design). This report documents every aspect of the "Bop-It" inspired game called "Dispose-It". In a team of 3, we built this device that prompts several items that the user has to dispose of correctly: either by "Trashing It", "Recycling It", or "Composting It". The paper explains details about the design and creation of the Software and Hardware (my part) units, as well as the enclosure. 

* Facial Recognition Lock System: Conducted during ECE1896 (Senior Design). This report documents every detail in the creation and testing of our facial recognition lock system. The device is able to recognize verified users and invalidate strangers with accuracies around 90%. I worked on the Computer Vision part involving the Machine Learning model used and a Liveness Detection system based on blink detection. More info under "projects" folder on my GitHub page!
